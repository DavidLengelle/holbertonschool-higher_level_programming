# Mini Agent Demo

Ce mini-projet montre une fonction de lancer de dés.

## Fonctionnalité

- `dice.js` exporte une fonction `roll(sides)`.
- `roll(sides)` retourne un entier aléatoire entre `1` et `sides`.
- `dice.test.js` appelle `roll(6)` cinq fois et affiche les résultats dans la console.

## Utilisation

Lancer le test avec Node.js :

```bash
node dice.test.js
```
